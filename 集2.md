在`_config.yml`中添加集合，就能通过`site.collections|size`看到添加了新集合

### 集合名使用中文
```yaml
collection6:
  name: 测试集合6
```
#### tag不能使用中文、不能使用空格
如果集合文件夹用中文命名（配置中当然也写中文集合名），
能通过`site.collections|size`看到添加了新集合，但新集合的任何信息都读不到。
```yaml
# 这些都无效
测试集合3:
  name: '测试集合3'
"测试集合4":
  name: '测试集合4'
'测试集合5':
  name: '测试集合5'
```

#### tag同样不能有空格
```yaml
collections:
  # 这些都无效
  test collection 7:
    name: '测试集合7'
  'test collection 8':
    name: '测试集合8'
```
